Section 1 Definitions

Apps

An app is a software application and unlike a website has to be downloaded by users.

Web Dev Trifecta

  • HTML - markup language used for structuring and displaying content
  • CSS - style sheet language that gives formatting and style to web page
  • JavaScript - scripting language that gives pages dynamic, interactive capabilities

HTML5 Structure Elements

These tell the computer the purpose of what you're making. For example, <nav> would tell the computer that the section is site navigation.
