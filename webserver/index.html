Vocabulary

Internet
A system of connected networks that link devices around the world
TCP/IP
Language devices use to connect to the internet
LAN
Connects devices in a small area like a campus or office
IP Address
A number given to every device connected to a network that uses Internet Protocol (IP)
HTTP
Rules for transfering files on the Web
Web Server
Delivers files that make Web pages to users
Web Client
User side of the Web
