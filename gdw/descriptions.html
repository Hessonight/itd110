HTML Terms

HTML
A common mark-up language
HTML element
A component of HTML, such and <title></title>
HTML tag
Represents an element in HTML
Angle Bracket
Used to enclose tags
HTML attribute
Used in open tag to control behaviour of an element
HTML comment
Information the webmaster can see but the ordinary user can't
HTML object element
Inserts objects such as images, don't have closing tags
HTML special character
Some characters have special uses in HTML, so if you want to use them literally you have to use special character. &lt makes <
